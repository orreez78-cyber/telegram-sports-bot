# 🤖 Telegram Sports Analytics Bot

Бот для анализа спортивных ставок с использованием искусственного интеллекта.

## 🏆 Поддерживаемые виды спорта

- ⚽️ **Футбол** — АПЛ, Ла Лига, Серия А, Бундеслига, РПЛ и другие
- 🏒 **Хоккей** — NHL, KHL, AHL, европейские лиги
- 🎮 **Киберспорт** — CS2, Dota 2, LoL

## ✨ Возможности

- 🧠 **Анализ матчей с помощью GPT-4** — бот сам считает вероятности, не копирует чужие прогнозы
- 📊 **Вероятностный анализ** — расчет шансов исходов в процентах
-  **Фильтр уверенности** — рекомендации только при шансе ≥ 60%
- ⏰ **Автоматическая рассылка** — ежедневные прогнозы в 10:00
- 💾 **База данных** — сохранение пользователей и прогнозов
-  **Кэширование** — экономия API запросов

## 🚀 Технологии

- **Python 3.11** — основной язык
- **aiogram 3.x** — Telegram Bot API
- **OpenAI GPT-4** — аналитика и прогнозы
- **SQLite** — база данных
- **APScheduler** — фоновые задачи
- **API-Football** — данные о футбольных матчах
- **TheSportsDB** — данные о хоккейных матчах

## 📦 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-username/ваш-репозиторий.git
cd ваш-репозиторий
```

### 2. Зависимости

```bash
pip install -r requirements.txt
```

### 3. Конфигурация (переменные окружения)

Секреты читаются только из окружения — в коде их больше нет. Скопируйте шаблон
и заполните его:

```bash
cp .env.example .env
# отредактируйте .env
```

Обязателен только `TELEGRAM_TOKEN`; остальные ключи включают отдельные источники
данных и деградируют мягко, если не заданы. **Любой ключ, ранее попадавший в git,
нужно перевыпустить.**

### 4. Запуск

```bash
python bot.py
```

## 📈 Бэктест и оценка качества

Модель проверяется без «подглядывания в будущее» (walk-forward): для каждого матча
прогноз строится только по данным до его начала, затем рейтинги обновляются.

```bash
python backtest.py --seasons 2021 2022 2023
```

Выводятся log-loss, Brier, точность и калибровка в сравнении с базовой моделью
(частоты исходов). Модуль `evaluation.py` содержит чистые функции метрик
(Brier, log-loss, калибровка, EV, Kelly, ROI).

## 💎 Value-беттинг

Обыграть маржу букмекера можно только ставками с положительным матожиданием:
ставка делается, лишь когда `вероятность_модели × коэффициент > 1`. Размер ставки —
дробный критерий Келли (¼) с жёстким потолком 5% банка. См.
`evaluation.select_value_bet`.
