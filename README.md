# 🤖 Telegram Sports Analytics Bot

Бот для анализа спортивных ставок с использованием искусственного интеллекта.

## 🏆 Поддерживаемые виды спорта

- ⚽️ **Футбол** — АПЛ, Ла Лига, Серия А, Бундеслига, РПЛ и другие
- 🏒 **Хоккей** — NHL, KHL, AHL, европейские лиги
- 🎮 **Киберспорт** — CS2, Dota 2, LoL

## ✨ Возможности

- 🧠 **Анализ матчей с помощью GPT-4** — бот сам считает вероятности, не копирует чужие прогнозы
- 📊 **Вероятностный анализ** — расчет шансов исходов в процентах
-  **Фильтр уверенности** — рекомендации только при шансе ≥ 60%
- ⏰ **Автоматическая рассылка** — ежедневные прогнозы в 10:00
- 💾 **База данных** — сохранение пользователей и прогнозов
-  **Кэширование** — экономия API запросов

## 🚀 Технологии

- **Python 3.11** — основной язык
- **aiogram 3.x** — Telegram Bot API
- **OpenAI GPT-4** — аналитика и прогнозы
- **SQLite** — база данных
- **APScheduler** — фоновые задачи
- **API-Football** — данные о футбольных матчах
- **TheSportsDB** — данные о хоккейных матчах
